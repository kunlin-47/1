# 尾矿库排洪系统失效致灾害链演化机理与断链防控研究

**摘要：** 尾矿库排洪系统失效是诱发尾矿库事故的重要致灾环节，其演化过程具有显著的链式传播特征。本文以国内1961—2025年间25起典型尾矿库排洪系统失效事故为研究样本，基于灾害链理论框架，构建了"固有因子—退化因子—触发因子"三元致灾因子体系及四种耦合机制模型。在此基础上，识别并定义了3类起始节点（N1—N3）、5类传播节点（P1—P5）与4类终态节点（T1—T4），归纳提炼出三条主灾害链及一条衍生次生链，并通过节点频率统计与路径转化概率分析对链式传播规律进行量化表征。研究表明：N2（排洪构件结构性破坏）是最主要的起始节点类型，在25起案例中出现频次占64%；T2（区域性生态污染）为最主要终态，占比84%；近年案例中T3（供水安全危机）占比持续上升，反映灾害链向社会系统延伸的演化趋势。据此，从预防性断链、工程性断链与应急性断链三个层次提出了靶向化防控建议，为尾矿库安全管理的精准化防控提供理论依据。

**关键词：** 尾矿库；排洪系统失效；灾害链；断链防控；节点体系

---

## 1 引言

尾矿库是用以堆存金属或非金属矿山进行选矿后排出的尾矿或其他工业废渣的场所，是矿山生产不可缺失的设施，同时也是一个具有高势能的人造泥石流危险源。排洪系统作为尾矿库防洪的核心工程屏障，承担着将洪水安全排出库外的关键功能。基于历史事故数据统计，国内尾矿库病害事故中排洪系统的病害事故占33.3%，我国有色金属矿山因排洪系统失事引起灾害的几乎占整个尾矿库事故的50%，排洪系统失效已成为尾矿库事故的主要诱发因素之一，其失效模式复杂多样，引发的灾害后果涵盖坝体溃决、大规模环境污染及公共安全事件等多种类型。

在尾矿库溃坝与失效研究领域，国内外学者已积累了较为丰富的成果。Rico等[2]对全球147起尾矿库溃坝事故进行统计分析，系统梳理了溃坝成因与后果的分布规律，发现排洪系统问题是重要的致灾因素之一。Bowker和Chambers[3,7]进一步指出，近年来尾矿库溃坝事故呈上升趋势，排洪系统能力不足和排水设计缺陷是尾矿库溢洪溃坝的关键致因。国内方面，郑欣等[9]对尾矿库溃坝事故进行统计分析，提出施工质量缺陷与极端气象是主要诱因；侯克鹏等[10]运用层次分析法(AHP)对尾矿库安全运行因素进行量化评估，定量证实排洪系统是决定整体安全的首要因素，其调洪与排洪能力的不足极易导致安全超高与干滩长度不达标进而引发溃坝；唐垒等[11]从极端条件下排洪系统失效视角分析了溃坝影响规律。

在灾害链理论方面，史培军等[7，8]提出灾害系统理论与灾害链研究框架，为本文提供了重要的理论基础。温家洪等[9]较早基于城市基础设施案例开展自然灾害连锁效应评价，揭示了链间转化与多链并发情景的研究价值；后续研究进一步将灾害链思想扩展到复合极端风暴洪水等多灾种复合情景的模拟与评估[10]。近年来，灾害链理论已被广泛应用于滑坡[11]、泥石流[12]、地震[13]等自然灾害及化工事故[14]等工业灾害分析。在尾矿库领域中，周科平等[15]将灾害链理论应用于尾矿库溃坝灾害的系统研究，提出"尾矿库溃坝灾害链及断链减灾控制"的研究框架。然而，现有研究多聚焦于"坝体失稳—泥石流—下游淹没"的宏观链式过程，针对排洪系统失效这一特定诱发因素所导致的灾害链演变机理，尚缺乏系统性的应用与研究。

综上所述，现有研究对尾矿库灾害的分析多聚焦于溃坝机理、坝体稳定性及整体风险评估等方面，鲜有研究从灾害链视角对排洪系统失效的链式传播机制进行系统性剖析，尤其缺乏对链间转化关系、节点转化概率与多链并发情景的定量化描述。本文以国内1961—2025年间25起典型尾矿库排洪系统失效事故案例为研究样本，构建排洪系统失效灾害链节点体系与传播模式；通过节点频率统计与路径转化概率分析对链式传播规律进行量化；基于断链减灾理论提出分层次靶向化防控建议，研究成果旨在为尾矿库安全管理的精准化防控提供理论参考。



## 2 排洪系统失效致灾因子及耦合机制分析

### 2.1 致灾因子识别与分类

基于灾害系统理论与自然-社会复合灾害分析框架，通常认为工程灾害的发生是致灾因子在承灾体上叠加作用的结果。针对尾矿库排洪系统这一特定承灾体，结合排洪系统工程特性与案例统计，本文将尾矿库排洪系统失效的致灾因子划分为"固有因子—退化因子—触发因子"三元体系。

固有因子是指系统在建设阶段已客观形成的先天性脆弱性；退化因子是指系统在运行阶段因物理、化学和管理过程而产生的渐进性劣化；触发因子是指在系统达到临界失稳状态时，诱发系统从隐性失效转变为显性事故的直接诱因。

三类致灾因子在时间维度上呈现出"先天脆弱性基础→渐进性能衰减→临界事件触发"的清晰演化层次。固有因子奠定脆弱性基础，退化因子推动系统性能持续趋近临界阈值，触发因子则在安全裕度耗尽时引发系统整体失效。三者并非独立作用，而是构成了因果递进的致灾逻辑链。

### 2.2 致灾因子耦合机制

尾矿库排洪系统失效通常并非由单一致灾因子独立触发，多因素相互作用与耦合演化才是导致系统性失效的主导机制。基于典型事故案例复盘与排洪系统运行特征，本文将致灾因子耦合机制归纳为四种基本类型。

**（1）串联耦合（链式递进型）**

固有因子、退化因子与触发因子按时序依次激活，风险具有明显的阶段性累积特征，为后期触发因子的作用提供条件基础。

**（2）并联耦合（多因素叠加型）**

多种致灾因子在同一时间或相近时间尺度内同步作用，通过叠加效应共同推动系统迅速达到或突破安全阈值。

**（3）正反馈耦合（自我强化型）**

系统初始局部结构损伤发生后，通过内部反馈机制不断强化损伤效应，形成"损伤—响应—再损伤"的循环过程，使系统状态向失效方向加速演化。

**（4）跨系统耦合（系统联动型）**

排洪系统失效突破单一工程系统边界，与坝体稳定系统、水环境系统及社会系统发生联动，形成多系统交互作用的级联灾害过程。

尾矿库排洪系统失效本质上是多类型致灾因子在复杂耦合关系下共同作用的结果。在演化过程中，各类耦合机制相互交织、交替主导，总体呈现"串联奠基—并联加速—正反馈突变—跨系统蔓延"的动力学演化规律，从系统层面深刻揭示了从隐性缺陷向显性灾变转化的内在机制。



## 3 排洪系统失效灾害链节点体系识别

### 3.1 灾害链节点的识别准则与层级架构

灾害链节点是灾害系统演化序列中的关键状态跃迁点，其本质是系统在多致灾因子耦合驱动下，从一种稳定态向另一种稳定态发生质变的临界时刻。节点不仅表征了系统状态的突变，更承载了灾害能量与物质的传递功能。基于灾害链演化动力学特征，本文确立了节点识别的三项核心准则：

（1）状态突变性：节点处系统状态发生质的转变，而非量的渐进变化；

（2）因果关联性：节点具有承上启下的逻辑功能，既是前置致灾过程的演化终点，又是后续灾害过程的触发起点。

（3）工程可识别性：节点状态在工程监测数据、物理现象表征或事后调查痕迹中具有明确的辨识特征，具备被观测与被度量的属性。

基于上述准则，运用系统分析法对25起典型尾矿库事故案例进行复盘推演，将排洪系统失效灾害链节点分为起始节点（N）、传播节点（P）与终态节点（T）三个层级，共识别出3类起始节点、5类传播节点与4类终态节点，构成完整的三层级灾害链节点体系。

### 3.2 灾害链节点体系构建与特征解析

起始节点表征致灾因子的激活与系统第一道防线突破的时刻；传播节点表征灾害能量的转化、传递与空间扩展过程中的关键状态；终态节点表征灾害后果的最终显现形式。各节点的物理定义与工程表征详见表1。

**表1 排洪系统失效灾害链节点体系及其特征解析**

| **节点层级** | **节点编码** | **节点名称** | **物理定义与工程表征** |
|:---:|:---:|:---|:---|
| **起始节点 (N)** | **N1** | 泄流能力失效 | 排洪系统实际过流能力低于设计洪水标准，表现为进水口违规抬高、淤积堵塞或汇水面积取值偏小等，导致排洪能力系统性不足。 |
| | **N2** | 排洪构件结构性破坏 | 排水井、涵洞、拱板等关键构件因施工质量缺陷、材料劣化或结构应力超限而发生断裂、坍塌或脱落，导致排洪通道物理阻断。 |
| | **N3** | 外部冲击性破坏 | 滑坡、泥石流等外部地质动力事件直接冲击排洪构筑物，导致排洪系统结构整体性丧失，为小概率高后果事件。 |
| **传播节点 (P)** | **P1** | 库区水位异常升高 | 库区水位加速上升，安全超高持续减小，浸润线抬高，表征水动力荷载的累积与漫顶风险的临近，是链式传播的核心中间节点。 |
| | **P2** | 坝体渗流通道形成 | 水流沿破损管道外壁或坝体裂隙形成集中渗流，产生持续水力侵蚀，是管涌破坏的前驱状态。 |
| | **P3** | 尾矿浆大量外泄 | 尾矿砂浆突破库区边界进入外部环境，标志灾害链由工程系统向环境系统跨越，是决定后续污染规模的关键传播节点。 |
| | **P4** | 坝体失稳征兆显现 | 坝坡发生局部滑动、异常沉陷或裂缝持续扩展，系统从可控渗流状态向结构性整体失稳转变的临界阶段。 |
| | **P5** | 污染物扩散至敏感水体 | 重金属等有害污染物随水体传输抵达饮用水源地或生态敏感区，灾害链向社会系统延伸的关键跨界节点。 |
| **终态节点 (T)** | **T1** | 坝体溃决 | 坝体结构完全失效，大量尾矿砂水混合物瞬间大规模释放，危害等级最高，往往造成严重人员伤亡与下游基础设施的毁灭性破坏。 |
| | **T2** | 区域性生态污染 | 重金属等持久性污染物在流域大范围扩散，导致水体长期超标，造成生态损害。 |
| | **T3** | 供水安全危机 | 污染物浓度超过水源地安全阈值，迫使下游水厂停供，引发公共卫生事件与社会恐慌。 |
| | **T4** | 尾矿泥石流 | 泄漏的尾矿浆在地形作用下演变为泥石流，形成速度快、冲击力强的次生地质灾害。 |

从节点统计特征来看，在起始节点分布中，N2（排洪构件结构性破坏）以64%的出现频次成为排洪系统失效最主要的触发类型，与施工质量管控不严密切相关；N1（泄流能力失效）占16%，主要源于设计参数偏低与违规运营；N3（外部冲击）占8%，属小概率高后果事件类型；多节点并发情形占12%，集中出现于超标暴雨叠加既有缺陷的复合情景中。

在终态节点分布中，T2（区域性生态污染）在25起案例中出现21次（84%），是最主要的灾害终态，反映排洪系统失效事故以环境污染为主要后果的整体特征。T1（坝体溃决）出现6次（24%），均伴有人员伤亡或重大财产损失；T3（供水安全危机）出现5次（20%），且全部集中于2015年以后，体现出灾害链向社会系统延伸的明显时代趋势。T4（尾矿泥石流）出现2次（8%），均为溃坝后的次生演化。

需要特别指出的是，P2（坝体渗流通道形成）在事故调查报告中未被直接记录。这并非表明该节点在实际演化中不存在，而是由其物理过程的隐蔽性决定的：管涌前驱阶段的集中渗流发生于坝体内部或管道外壁土体中，在事故发生前难以被现有监测手段捕获，事故后又被更显著的终态破坏掩盖。从工程力学机理来看，构件破损后水流沿管壁外侧形成优势渗流通道是坝体结构劣化的必然中间过程，P2节点的设置基于渗流力学理论推演与已有尾矿库管涌破坏研究成果，其工程意义在于为预警监测提供靶向识别目标。



## 4 灾害链链式传播模式与机制分析

基于灾害链节点体系识别结果，结合25起典型事故案例的演化路径分析，本文归纳提炼出三条主灾害链及一条衍生次生链。三条主链分别对应排洪系统失效的三种核心演化模式，衍生次生链为主链演化的后续延伸。各链通过共享节点形成内在关联，是灾害后果非线性放大的结构性根源。

### 4.1 主链Ⅰ：泄流能力失效→洪水漫顶→溃坝灾害链

主链Ⅰ描述排洪系统因泄流能力不足（N1），导致库区水位持续异常升高（P1），最终引发洪水漫顶并致坝体溃决（T1）的演化过程。演化路径为：

**N1 → P1 → 洪水漫顶 → T1**

主链Ⅰ的传播动力在于水动力能量的持续累积。从致灾因子耦合视角分析，N1的形成通常源于固有因子（设计参数偏低、进水口违规抬高）与触发因子（超标洪水）的并联耦合。在25起案例中，以N1为起始节点的案例造成了最为突出的人员伤亡（案例5致49人死亡、案例16致22人死亡），印证了该链向T1演化的高致死性特征。P1节点的实时监测与预警是阻断该链向T1演化的关键干预点。

### 4.2 主链Ⅱ：排洪构件结构破坏→坝体失稳→溃坝灾害链

主链Ⅱ描述排洪构件因施工质量缺陷、材料劣化或外力冲击发生结构性破坏（N2），进而引发坝体失稳并最终溃决（T1）的演化过程。主链Ⅱ存在两条相互竞争的演化路径：

**路径A**（水位上升主导）：N2（构件破坏致排洪通道受阻）→ P1（库水位持续升高）→ P4（坝体失稳征兆）→ T1（溃坝）

**路径B**（渗流侵蚀主导）：N2（构件破坏形成通道缺口）→ P2（坝体集中渗流通道形成）→ P4（管涌侵蚀加速坝体失稳）→ T1（溃坝）

两条路径可在同一事故中并发演化，相互加速，显著压缩预警与处置时间窗口。路径A以超静水压力为主要驱动力，路径B以渗流水力梯度为主要驱动力。

从25起样本的统计分析来看，主链Ⅱ呈现出"理论完备但显性案例占比较低"的特征，这一现象可从以下三方面加以解释：

**第一，主链Ⅱ常被主链Ⅲ"竞争性掩盖"。** 当N2激活后，构件破损在工程实践中更多表现为"通道性泄漏"（即管道或井体破口直接成为尾矿外泄通道），而非"堵塞性失效"（即排洪通道完全阻断导致水位持续升高）。前者直接激活主链Ⅲ（N2→P3→T2），后者方能激活主链Ⅱ。在25起案例中，N2起始后演化为主链Ⅲ的路径占据主导地位，这反映了排洪构件破坏的破损形态以局部穿透性破裂为主的工程现实。

**第二，主链Ⅱ的路径B中P2节点具有强隐蔽性。** P2（坝体渗流通道形成）作为管涌破坏的前驱状态，其物理过程发生于坝体内部或管道外壁土体中，在事故调查中极难被追溯识别。案例2（河北白庙子）中"钢筋混凝土排水管道沉陷、错位、断裂"后发展至"堵塞溢洪沟并冲毁堤坝"的过程，以及案例11（夜长坪）中"排水斜槽局部坍塌"后形成"直径80~100m漏斗"的演化轨迹，均隐含P2→P4的渗流侵蚀传播过程，只是在事故报告中未被单独记录。

**第三，主链Ⅱ代表最高风险等级的演化通道。** 虽然其在样本中的显性频次低于主链Ⅲ，但一旦完整激活则必然导向T1（坝体溃决）这一最严重终态。从风险管理角度，主链Ⅱ属于"低频率—高后果"类型，其防控优先级不应因统计频次较低而被降低，恰恰相反，P2节点的早期识别（如坝体渗压异常监测）是阻断该链向T1演化的关键"断链窗口"。

### 4.3 主链Ⅲ：排洪构件破损→尾矿直接泄漏→环境污染灾害链

主链Ⅲ描述排洪构件发生局部破损（N2/N3），导致尾矿砂浆突破库区边界直接外泄（P3），并在流域尺度扩散至敏感水体（P5），最终形成区域性生态污染（T2）或供水安全危机（T3）的演化过程。演化路径为：

**N2/N3 → P3 → P5 → T2/T3**

该链区别于主链Ⅰ、主链Ⅱ的核心特征在于：其终态节点并非坝体结构破坏，而是污染物质的跨界扩散。主链Ⅲ是25起案例中激活频次最高的灾害链路径，在16起N2起始的案例中有14起（87.5%）沿此链演化，在大型流域中影响距离可达数百千米（案例20甘肃陇星锑业影响346 km河道、案例22黑龙江鹿鸣影响约340 km河道），且影响具有持续性和难逆转性。

该链在近年事故中占比持续上升，P5节点的识别依赖于流域水质自动监测站网的实时响应能力，预警时间的长短直接决定下游水厂的应急处置效果。值得注意的是，主链Ⅲ演化过程中存在"断链成功"的典型案例（案例9广西跃华——被下游库截留；案例10江西大茅山——缺口及时封堵；案例23河南鑫德——9天应急截污），表明该链在P3→P5区间的可干预性较强，这为工程性断链与应急性断链的实施提供了实证支撑。

### 4.4 衍生次生链：大规模泄漏→尾矿泥石流→次生灾害链

衍生次生链是主链Ⅰ或主链Ⅱ演化至终态节点T1（坝体溃决）后的后续延伸，描述大规模外泄尾矿在沟谷地形与重力作用下演变为尾矿泥石流（T4）的次生灾害过程。该链的激活条件包括：库内蓄存尾矿量充足、下游为具有汇流效应的沟谷地形以及尾矿砂水比处于泥石流可运移的临界范围。

衍生次生链一旦激活，灾害影响范围由库区扩展至整个下游流域，破坏机制由"溃坝洪水"转变为"固液两相流冲击"，具有流速快、冲击力强、淤积范围广的特点，使灾害由"工程事故"演变为"流域性地质灾害"，是尾矿库灾害中危害范围最大的终态形态。在25起案例中，案例5（湖南牛角垄，49人死亡、冲毁房屋39栋）和案例16（广东信宜紫金，22人死亡、523户房屋全倒）为该链的典型代表。

### 5.2 共享节点与起始-终态联合分布

三条主灾害链在实际事故中并非相互独立的平行路径，而是呈现高度的共生性与转化性。从共享节点角度分析：

**N2是主链Ⅱ与主链Ⅲ的共同起始节点：** 同一构件失效可同步开启两类截然不同的演化路径。若失效主要导致排洪通道堵塞，则激活以溃坝为终态的主链Ⅱ；若失效主要导致库区边界破损，则激活以环境污染为终态的主链Ⅲ。两条路径的竞争关系取决于构件破损的空间位置与破坏形态。

**P1是主链Ⅰ的核心中间节点，** 同时在主链Ⅱ路径A中扮演传播中枢角色，其物理意义在于水动力荷载的累积状态表征。

**T1是主链Ⅰ与主链Ⅱ的共同终点，** 并进一步向衍生次生链延伸，形成"溃坝→泥石流"的序列次生灾害结构。T1节点的触发意味着灾害链进入不可逆演化阶段，后续灾害的规模完全由库存量与地形条件决定。

（2）**N2→T2为绝对主导通道：** 在19起N2起始案例中，17起（89.5%）激活T2终态，反映排洪构件结构性破坏以环境污染为主要后果模式（对应主链Ⅲ主导）。其中4起同步激活T3，表明部分案例存在T2→T3的链内升级。

（3）**N2→T1通道（主链Ⅱ）虽频次较低但不可忽视：** 2起案例（案例2河北白庙子、案例11夜长坪部分激活）体现了主链Ⅱ的演化特征，加之案例11虽被成功断链但已发展至P4阶段，表明主链Ⅱ路径在工程条件恶化时仍具有向T1完整演化的潜力。

（4）**N3属"小概率—高后果"类型：** 仅2起案例，但均直接造成人员伤亡，验证了外部冲击性破坏的突发性与不可预测性。

（2）**P3→P5的跨界转化概率为52.9%**，表明尾矿外泄后有超过半数的概率扩散至敏感水体，这一较高的转化率揭示了主链Ⅲ向社会系统延伸的强倾向性，也凸显了P3→P5区间作为关键断链窗口的工程价值。

（3）**主链Ⅲ以64%的整体激活概率成为排洪系统失效灾害链中的绝对优势路径**，而主链Ⅱ显性激活概率仅为8%。但从风险矩阵视角分析，主链Ⅱ虽频率低但后果严重度极高（必然指向T1），其风险等级并不低于高频率的主链Ⅲ。

链间升级转化是指某一灾害链在特定条件下演化升级为危害程度更高的另一条灾害链的动态过程，是灾害后果非线性放大的关键机制。主要转化路径包括：

**（1）转化路径1（主链Ⅲ→主链Ⅰ）：** 排洪构件破损引发尾矿直接泄漏（主链Ⅲ激活），若同步造成排洪通道完全堵塞，则进一步导致库水位持续升高，激活主链Ⅰ的漫顶演化路径。其转化条件为：构件破损程度既要足以引发尾矿泄漏，又要能堵住过水通道触发水位升高，对应并联耦合向串联耦合的机制转变，灾害后果可能由环境污染升级为溃坝人员伤亡。

**（2）转化路径2（主链Ⅲ→主链Ⅱ）：** 管道断裂引发尾矿泄漏（主链Ⅲ激活）的同时，尾砂沿坝体内部形成集中渗流通道，逐步发展为管涌侵蚀，激活主链Ⅱ的坝体失稳路径。该转化对应正反馈耦合机制，渗流侵蚀与管道破损相互促进，系统劣化速度显著加快，处置时间窗口大幅压缩。

**（3）转化路径3（主链Ⅰ/Ⅱ→衍生次生链）：** 主链Ⅰ或主链Ⅱ演化至终态节点T1（坝体溃决）后，若库内尾矿量充足且下游为沟谷地形，便会自动触发衍生次生链（尾矿泥石流）。此时灾害影响范围由库区扩展至整个下游流域，由"工程事故"转变为"流域性地质灾害"。

### 5.4 多链并发情景分析

在极端气象条件（超标暴雨）与排洪系统既有结构缺陷叠加的复合情景下，多条灾害链可能被同步激活，形成多链并发的复合灾害情景。案例16（广东信宜紫金）是多链并发的典型代表：排水井进水口被擅自抬高（N1特征）+ 拱板未按要求打开（N2特征）→ 在台风暴雨触发下同步激活主链Ⅰ（泄流不足→水位升高→漫顶）与主链Ⅱ（结构约束失效→坝体整体失稳）→ 最终溃坝（T1）并衍生泥石流（T4），造成22人死亡。

这一情景的内在驱动机制在于超标暴雨不仅直接提供了激活主链Ⅰ（N1→P1）的水动力条件，其附带的强大水力冲刷力同时对既有结构缺陷（如拱板开裂、管道锈蚀）施加超限荷载，进而触发N2节点，激活主链Ⅱ与主链Ⅲ。三链并发→相互加速→共同收敛于T1→衍生T4的级联结构，是尾矿库灾害中破坏力最强、响应难度最大的复合情景。

从时间维度分析，多链并发情景下灾害链演化速度最快、干预时间窗口最短。相关典型案例表明，在此类复合情景中，从起始节点激活到坝体溃决的时间可短至数小时至十余小时。多链并发情景对应致灾因子耦合机制中的"并联耦合→正反馈耦合"快速转化过程，是尾矿库汛期安全管控中风险等级最高的情景类型，应作为应急预案编制的优先设计工况。

### 5.5 断链成功案例分析与启示

在25起案例中，有3起实现了完全断链（案例9、10、11），1起实现了部分断链（案例23的T3被阻断）。断链案例的系统分析对防控策略具有直接指导价值，具体见表5。

**表5 断链成功案例分析**

| **案例** | **已演化至** | **断链位置** | **阻断机制** | **断链类型** |
|:---|:---:|:---:|:---|:---|
| 案例9 广西跃华 | P3 | P3→T2之间 | 外泄尾砂被下游鸿塔尾矿库全部截留 | 被动工程屏障型 |
| 案例10 江西大茅山 | P3 | P3→T2之间 | 缺口3~5m时现场及时封堵 | 快速应急响应型 |
| 案例11 河南夜长坪 | P4 | P4→T1之间 | 漏斗坑形成后工程干预，未演化为溃坝 | 工程加固阻断型 |
| 案例23 河南鑫德 | P5 | P5→T3之间 | 9天应急处置将污染控制于德亭河 | 流域应急截污型 |

断链案例分析揭示以下规律：（1）**所有断链均发生在传播节点（P）阶段**，无一例发生在终态节点之后，表明P节点是灾害链防控的"黄金干预窗口"；（2）断链成功率与发现-响应时间密切相关，案例10在缺口仅3~5m时即实施封堵，代表了"早发现—快响应"的理想模式；（3）被动工程屏障（如案例9的串联库体系）无需人为决策即可自动实现断链，是鲁棒性最强的防控机制。



## 6 基于断链理论的防控启示

断链减灾理论指出，灾害链防控的核心在于精准识别并切断链式演化过程中的关键节点，致使灾害能量在跨阶段传递时发生衰减或阻断。基于前文构建的三条主链节点体系与传播机制，本研究从预防性断链、工程性断链与应急性断链三个层面提出防控建议，旨在构建"源头削弱—过程阻滞—末端衰减"的三位一体灾害链防控体系。

### 6.1 预防性断链

预防性断链着眼于从源头切断N1、N2的形成条件，是灾害链防控效率最高、成本最低的干预层级，主要措施包括：

（1）**严格水文校核与防洪达标评估：** 严格执行排洪系统水文参数核查，尤其关注汇水面积的准确取值，避免设计防洪标准系统性偏低，从源头消除N1节点的先天形成条件。针对已建库的设计参数复核，应定期开展防洪能力达标评估。鉴于N1→T1转化概率高达100%，预防N1的形成具有最高的防控优先级。

（2）**强化施工质量管控：** 针对拱板、涵洞衬砌、井架基础等隐蔽工程，推行全过程质量追溯机制，重点检查配筋率、混凝土强度及接缝处理质量，消除施工质量缺陷这一最普遍的固有因子。案例20（甘肃陇星锑业）和案例22（黑龙江鹿鸣）的事故调查均明确指出拱板施工质量不达标为直接原因，凸显了该措施的针对性。

（3）**规范运营期人为活动：** 建立进水口位置定期核查制度，严禁违规抬高进水构件等改变原设计水力条件的行为。案例16中进水口被擅自抬高2.597m的违规行为直接削弱了排洪能力约40%，是导致多链并发灾害的直接人为因素。

### 6.2 工程性断链

工程性断链着眼于在P1—P5传播节点处阻断灾害链的进一步演化，通过物理或技术手段植入"阻断器"，截断灾害能量的传递通道。主要措施包括：

（1）**实施库水位动态控制（靶向P1节点）：** 完善实时库水位与安全超高监测预警体系。在汛期实施前置性的库容腾空调度，确保P1节点被激活时具备充足的水位调控冗余，截断其向漫顶溃决状态演化的路径。鉴于P(T1|P1)=100%的统计特征，建议将安全超高的动态阈值作为"一票否决式"预警指标。

（2）**强化坝体渗流监测与工程防渗措施（靶向P2节点）：** 合理布设坝体渗流压力与位移监测系统，实时掌握P2、P4节点的状态变化。虽然P2节点在事故报告中难以被直接观测，但通过渗压计异常波动、坝趾渗水浑浊度变化等间接指标可实现早期预警。对已出现集中渗流异常的区域，及时采取帷幕灌浆等工程措施，切断渗流破坏的演化路径，防止主链Ⅱ从P2向P4→T1方向演化。

（3）**下游预置污染扩散阻控工程（靶向P3→P5区间）：** 在库区下游敏感水体上游预置应急截污坝或拦污网，在P3节点激活后快速启动，阻断外泄尾矿进入重要水体的扩散路径。案例9（广西跃华）的断链成功证实，串联库体系可有效截留外泄尾砂，实现P3→T2路径的自动阻断。建议在有条件的矿区推广"下游缓冲库"或"事故应急池"设计理念。

### 6.3 应急性断链

应急性断链着眼于在灾害链已演化至终态节点时，通过快速应急响应，最大限度压缩终态危害的影响范围，主要措施包括：

（1）**完善溃坝应急预案与靶向疏散体系：** 基于溃坝洪水演进数值模拟结果绘制高精度淹没风险图，确保在T1节点触发时能即时启动人员靶向疏散；同步构建针对尾矿泥石流（T4）演进路径的专项应急处置预案，遏制次生地质灾害的叠加破坏效应。统计表明P(T4|T1)=33.3%，溃坝后伴生泥石流的概率不容忽视。

（2）**构建流域水环境应急监测与预报网络（靶向P5→T3区间）：** 在下游流域梯级布设水质自动监测站网，实现对P5→T3演化过程的实时追踪与浓度预测，为下游水厂采取停水等应急措施预留预警时间。案例23（河南鑫德）通过9天持续应急处置将污染控制在德亭河段，成功阻断了T3的激活，验证了流域应急监测网的实效性。统计表明P(T3|P5)=44.4%，接近半数的P5节点激活会发展为供水危机，凸显了该监测网络建设的紧迫性。

（3）**建立跨部门协同应急响应机制：** 建立矿山企业—环保部门—应急管理部门—水务部门的跨系统联动机制与信息共享平台。当灾害链突破单一系统边界，进入跨系统耦合阶段（P5节点），确保各部门快速协同响应，避免因信息滞后延误最佳处置时机。这是应对主链Ⅲ及衍生次生链的关键制度性保障。



## 7 结论

本文基于灾害链理论，以25起典型尾矿库排洪系统失效事故为研究样本，系统构建了排洪系统失效灾害链的致灾因子体系、节点层级框架与链式传播模式，深入剖析了链间耦合转化机制与多链并发情景，并提出了三层次靶向防控策略。主要研究结论如下：

（1）"固有因子—退化因子—触发因子"三元体系较为完整覆盖了排洪系统失效的致灾因子空间，四种耦合机制（串联、并联、正反馈、跨系统）揭示了从隐性缺陷向显性灾变转化的动力学路径。其中，施工质量缺陷（固有因子）与超标降雨（触发因子）的并联耦合是案例中出现频次最高的耦合模式。

（2）识别并定义了3类起始节点（N1—N3）、5类传播节点（P1—P5）与4类终态节点（T1—T4），形成具有工程可操作性的灾害链节点识别与监测靶向体系。N2（构件结构性破坏）在统计案例中出现频次最高（64%），T2、T3终态在近年案例中占比呈上升趋势，揭示了灾害链向社会系统延伸的演化特征。其中，P2（坝体渗流通道形成）作为理论推演的隐性传播节点，虽在事故报告中未被直接记录，但其工程力学机理明确、预警监测价值突出，为主链Ⅱ的早期断链提供了关键靶向识别目标。

（3）三条主链（漫顶溃坝链、坝体失稳溃坝链、直接泄漏污染链）通过共享节点N2、P1和T1形成内在耦合关系。主链Ⅲ（N2→P3→P5→T2/T3）以64%的整体激活概率成为绝对优势路径；主链Ⅰ（N1→P1→T1）虽频次较低但P1→T1转化概率达100%，具有最高致死风险；主链Ⅱ（N2→P2→P4→T1）属"低频率—高后果"型，其显性案例虽少但理论演化路径完备，在工程条件恶化时具有向T1完整演化的潜力。三条主链的风险特征差异化明显，防控策略应针对性匹配。

（4）极端气象与既有缺陷叠加的多链并发情景，是尾矿库汛期安全管控中风险等级最高的极端场景，应作为应急预案编制的优先设计工况。断链案例分析表明，所有成功断链均发生在传播节点（P）阶段，P3→P5区间是主链Ⅲ可干预性最强的"黄金窗口"，被动工程屏障型断链（如串联库截留）具有最高鲁棒性。

（5）"预防性断链（N节点）—工程性断链（P节点）—应急性断链（T节点）"的三层次体系与灾害链节点结构精准对应，为尾矿库分层、分级、靶向化安全管控提供了可操作的理论框架。

本文的局限性在于：受公开数据可及性限制，25起案例的事故信息完整程度参差不齐，部分节点（尤其是P2）的识别依赖于工程力学推演而非直接观测数据，节点转化概率的统计精度仍有待更大样本数据的进一步验证；灾害链中各传播过程的物理机制尚需结合数值模拟方法进行定量深化。未来研究可引入贝叶斯网络或事故树等方法对节点转化概率进行更严格的概率量化，并将案例库进一步扩充以提高统计结论的可靠性；同时，P2节点的工程监测指标体系与阈值确定将是后续重点研究方向。
