## 等级 1：必改（不改将面临直接拒稿风险）

### 1.1 删除/弱化所有 100% 类的频率断言

**问题**：`F(P1|N1)=100%`、`F(P3|N3)=100%`、`F(T1|P1)=100%`、`F(T1|N3)=100%` 这四处 100% 都来自 n=2 或 n=3 的极小样本。任何有统计学背景的审稿人会立即指出"小样本下的 100% 不构成统计推断"。

**改法（任选其一）：**

**方案 A（最稳妥，推荐）**：保留频率值，但增加 Wilson 95% 置信区间列，并在解读时使用方向性表述

把原表 4 改造成下表（已包含计算结果，可直接套用）：

| 演化阶段 | 概率指标 | 物理意义 | 样本计数 | 频率点估计 | **Wilson 95% CI** |
|---|---|---|---:|---:|:---:|
| **A. 起始→传播阶段** | F(P1\|N1) | N1 引发库区水位异常升高 | 3/3 | 100.0% | **[43.8%, 100.0%]** |
|  | F(P3\|N2) | N2 引发尾矿浆外泄 | 19/20 | 95.0% | **[76.4%, 99.1%]** |
|  | F(P4\|N2) | N2 引发坝体失稳征兆 | 2/20 | 10.0% | **[2.8%, 30.1%]** |
|  | F(P3\|N3) | N3 引发系统损伤及外泄 | 2/2 | 100.0% | **[34.2%, 100.0%]** |
| **B. 传播链扩散与阻断** | F(P5\|P3) | P3 后跨界扩散至敏感水体 | 9/21 | 42.9% | **[24.5%, 63.5%]** |
|  | F(完全断链\|P3) | P3 后所有终态均未激活 | 2/21 | 9.5% | **[2.7%, 28.9%]** |
|  | F(任何成功干预\|P3) | P3 后受到工程/应急干预 | 3/21 | 14.3% | **[5.0%, 34.6%]** |
| **C. 传播→终态阶段** | F(T1\|P1) | P1 最终导致漫顶溃坝 | 3/3 | 100.0% | **[43.8%, 100.0%]** |
|  | F(T1\|N3) | N3 直接导致突发性溃坝 | 2/2 | 100.0% | **[34.2%, 100.0%]** |
|  | F(T2\|P3) | P3 后造成区域生态污染 | 17/21 | 81.0% | **[60.0%, 92.3%]** |
|  | F(T3\|P5) | P5 后实际酿成供水安全危机 | 3/9 | 33.3% | **[12.1%, 64.6%]** |
|  | F(T3阻断\|P5) | P5 后供水危机被应急阻断 | 1/9 | 11.1% | **[2.0%, 43.5%]** |
|  | F(T4\|T1, N1路径) | N1 路径溃坝后衍生伴生尾矿泥石流 | 2/3 | 66.7% | **[20.8%, 93.9%]** |

并在表格下方添加一段说明：

> **注**：考虑到本研究 25 例样本规模较小，部分条件概率分子分母均较小（如 F(P1\|N1)=3/3、F(P3\|N3)=2/2）。为客观反映估计的不确定性，本表对每一指标同步给出基于二项分布的 Wilson 95% 置信区间。Wilson 区间在小样本及频率为 0 或 1 的边界值情形下表现稳健，是小样本条件概率估计的推荐方法。**频率为 100% 的指标（如 F(P1\|N1)、F(T1\|P1)）由于样本量极小（n≤3），其置信区间下界仅为 34%–44%，仅能支持"该路径在已观测案例中均被激活"的方向性结论，不构成对路径转化必然性的概率断言。**

**方案 B（更保守）**：直接将 100% 的指标从概率表中移除，改用文字描述

例如把 F(P1|N1)=100% 改写为：

> "在 3 例 N1 起始的样本案例中，均观察到 P1 节点（库区水位异常升高）的激活，反映 N1→P1 是该类起始路径的主导演化方向。"

> "受样本规模限制（n=3），本研究不对 N1→P1 转化路径给出概率估计。"

### 1.2 第 5.2 节"节点转化概率与链间升级路径"段落语言保守化

**原文段落**：

> "（1）起始阶段的确定性与路径分化：基于本次研究的统计样本，N1（泄流能力失效）100%引发库区水位异常升高（P1），而N3（外部冲击）则100%直接引发尾矿浆外泄（P3）。"

**改后段落**：

> "（1）起始阶段呈现明确的路径分化特征：在样本案例中，N1（泄流能力失效）3/3 案例引发库区水位异常升高（P1），N3（外部冲击）2/2 案例引发尾矿浆外泄（P3）；尽管这两个比例的置信区间下界分别为 43.8% 与 34.2%，但所有已观测案例均沿同一路径演化的事实，仍体现了起始阶段路径分化的方向性。"

### 1.3 删除/改写所有"100%"作为名词使用的位置

通读 1.md，把以下表达全部替换：

| 原文 | 改后 |
|---|---|
| "N1 …… 100%引发……" | "N1 起始的样本案例中，全部 3 例引发……" |
| "N3 …… 100%直接引发……" | "N3 起始的样本案例中，全部 2 例直接引发……" |
| "P1 100%导致漫顶溃坝" | "P1 节点激活后，3 例样本均演化至漫顶溃坝（T1）" |

**操作要点**：用搜索功能查 "100%"，逐处改写。

### 1.4 弱化"持续上升"等趋势断言

**原文段落**：

> "近年案例中T3（供水安全危机）占比持续上升，反映灾害链向社会系统延伸的演化趋势"

**问题**：T3 仅 3 例（陇星锑业 2015、鹿鸣矿业 2020、嵩县鑫德 2024，其中嵩县鑫德 T3 被成功阻断），用"持续上升"作为趋势断言，证据偏弱。

**改后段落**：

> "T3（供水安全危机）类终态在 2015 年前的 18 例样本中均未出现，而在 2015 年后的 7 例样本中出现 3 次（占该子样本约 42.9%，其中 1 例被成功应急阻断）。这一对比初步显示出案例向社会系统延伸的迹象，但由于近年子样本量较小，该现象是否构成长期趋势仍需后续更大样本进一步验证。"

### 1.5 第 5.3 节多链并发情景需要明确"理论工况"定位

**原文段落**：

> "三链并发→相互加速→共同收敛于 T1→衍生 T4 的级联结构，是尾矿库灾害中破坏力最强、响应难度最大的复合情景。"

**问题**：原文紧接着用"相关典型案例表明"引出量化（"从起始节点激活到坝体溃决的时间可短至数小时至十余小时"），但 25 例样本中并未真正观测到三链同步激活的案例。

**改后段落**：

> "在极端复合情景的理论分析中，三链并发可能形成'相互加速→共同收敛于 T1→衍生 T4'的级联结构，这是尾矿库灾害中理论上破坏力最强、响应难度最大的工况类型。本研究 25 例样本中未直接观测到严格意义上多主链同步激活的案例，但部分多终态组合（如 T1+T4）反映了主链向衍生次生链的纵向延伸过程。多链并发情景的发生概率与时序特征，受限于样本规模与事故复盘信息完整度，本文不对其进行经验频率估计，建议在未来研究中通过扩展案例库与物理力学推演加以量化。"

## 等级 2：强烈建议改（直接影响审稿人评分）

### 2.1 在第 7 章"结论"或单独的"研究局限"段落中诚实说明限制

在第 7 章末尾或新增一个 "局限性与未来工作" 子节，加入：

> "本研究的方法论局限主要体现在三方面：(1) 样本规模限制——25 例样本中部分起始节点（N1、N3）的子样本量仅 2–3 例，相应条件概率估计的置信区间偏宽，仅能支持方向性结论而非精确概率断言；(2) 时间异质性——样本跨越 1961–2025 年共 64 年，期间尾矿库设计规范、施工技术、监测能力均发生显著变化，本研究将所有案例置于同一频率分母统计，隐含'条件概率时间稳定'假设，该假设有待后续分时段研究检验；(3) 隐蔽节点观测——P2（坝体集中渗流通道）受其物理隐蔽性限制，在事故调查报告中未被直接记录，本研究对该节点的设置基于渗流力学推演，相应的转化概率估计精度有限。未来研究可通过扩展案例库（含国际案例）、引入贝叶斯方法对节点转化概率进行更稳健的量化，并结合数值模拟对隐蔽节点的物理过程进行精细刻画。"

### 2.2 在引言或第 5 章开头明确说明本研究的统计推断属性

在第 5 章开头或第 1 章末尾加一段：

> "需要说明，本研究的概率统计采用案例频率法，给出的条件概率为对样本案例演化路径的描述性统计而非预测性概率。具体而言，频率值 F(B\|A) 的解释为'在 A 节点已激活的样本子集中，B 节点同步激活的案例比例'，并不直接外推为'未来类似条件下 A 激活后 B 必然激活的概率'。后者的严格量化需要更大样本规模与基于专家先验的贝叶斯推断方法。"

### 2.3 P2 节点的相关论述要更明确"理论推演"属性

**原文段落**：

> "需要特别指出的是，P2（坝体渗流通道形成）在事故调查报告中未被直接记录。本研究认为，这一现象并非表明该节点在实际演化中不存在……"

**改后段落**：

> "需要特别指出的是，P2（坝体渗流通道形成）在 25 例事故调查报告中均未被直接记录。**本研究将 P2 列入节点体系，主要基于渗流力学的理论推演而非样本观测**。这一处理具有两方面含义：(1) P2 在节点体系中的存在性主要由工程力学机理论证支撑，而非由案例频率验证；(2) 涉及 P2 的转化概率（如 F(T1|P2)）在本研究中未给出经验估计，相关讨论限定于演化机理层面。P2 节点的设置目的是为预警监测提供靶向识别目标，而非作为已观测的统计单元。"

### 2.4 摘要中删除任何与表 4 中"100%"对应的强化表述

通读摘要，确认没有"……必然引发……"或"100% 演化为……"类的表述。当前摘要内容相对克制，但建议在"对链式传播规律进行量化表征"一句后加：

> "……量化表征，并在表征过程中通过引入 Wilson 置信区间显式刻画样本规模导致的统计不确定性。"

## 等级 3：可选改进（锦上添花，不影响发刊门槛）

### 3.1 增加 1–3 个 2015 年后案例（如能找到可信来源）

**优先类型**：

- **N1 起始的近期案例**（你的 N1 子样本只有 3 个，最近的是 2010 年高旗岭，确实偏老）
- **成功断链的案例**（你只有 3 例，扩展到 4–5 例即可显著强化"断链可干预性"论述）
- **T3 类终态案例**（仅 3 例，强化"近年向社会系统延伸"的论据）

**可信来源（必须自行核实，不可凭二手信息直接引用）**：

1. **国家矿山安全监察局**官网事故通报栏目：[chinamine-safety.gov.cn](https://www.chinamine-safety.gov.cn/)
2. **生态环境部** "突发环境事件信息发布"：[mee.gov.cn](https://www.mee.gov.cn/) 检索 "尾矿"
3. **各省应急管理厅** 年度安全生产报告（重点：陕西、河南、湖南、广西、云南、内蒙古）
4. **CNKI 二手综述**：检索"尾矿库 + 排洪 + 事故案例"，参考王云海、郑欣、Lyu Z 等人的统计文献
5. **WISE-Uranium 全球数据库**（含中国部分）：[wise-uranium.org/mdaf.html](https://www.wise-uranium.org/mdaf.html)
6. **GRID-Arendal 全球尾矿库事故数据集**

**操作建议**：

- 每个候选案例必须满足：
  (a) 有政府/监管部门或同行评议期刊的一手报告；
  (b) 排洪系统失效是直接致灾环节；
  (c) 包含起始节点、传播节点、终态节点的清晰描述。
- 不满足上述任一条件的案例**宁可不加**。

### 3.2 增加 1–2 个国际典型案例做对比验证（可选）

虽然小论文以国内案例为主，但加 1–2 个国际典型案例做"链式结构具有跨国普适性"的论证，可显著提升论文档次。推荐：

- **Stava 1985（意大利）**：排水井渗流引发坝体破坏，主链Ⅱ典型
- **Aurul 2000（罗马尼亚）**：跨国污染事件，主链Ⅲ→T3 国际样本
- **Brumadinho 2019（巴西）**：现代典型案例，资料完整

加在哪里：可作为表 1 的扩展，或在第 4 章某处简单引用对比。**不建议大幅扩展，1–2 段即可**。

### 3.3 表格视觉优化

## 一句话原则

**这份小论文的发刊路径不是"加更多案例"，而是"把现有 25 个案例的统计语言写得更诚实"。**

后者改动小、风险小、效果好；前者投入大、不确定性高、回报有限。

把精力放在审稿人最容易命中的 4 类硬伤上（100% 的伪精确、趋势断言过强、多链并发证据缺失、研究局限不诚实），就足以让这篇文章达到发刊水平。

## 关于"补 8–9 个 2015 年后案例"的最终建议

- **如果你时间紧（1–2 周内要交导师）**：**不要补**。按本清单等级 1+2 修改即可达到发刊门槛。100% 这类硬伤一旦修掉，审稿人最大的把柄就消失了。
- **如果你时间充裕（1 个月以上）**：**补 2–3 个 2015 年后案例**即可，不必硬凑 8–9 个。重点补 N1 起始的近期案例和 T3 类终态案例。
- **不要做的事**：不要为了凑数把不完整、不可靠、来源不清的案例硬塞进案例表——单个不可靠案例的损害大于多个案例的边际收益。
